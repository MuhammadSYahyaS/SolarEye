SolarEye build May 31, 2017
Gemeinschaft Team
---------------------------
A small software that could do object recognition of the sun using Gradient Hough Transform in OpenCV library and angular deviation computation of the sun's position relative to the center of the frame from a video.
---------------------------
How to use :
1. Install VC 2013 Redistributable. Included in Support folder (vcredist_x86.exe).
2. Make sure that .NET Framework 4.0 is already installed. You can get it from Microsoft webpage.
3. Install WINGDNG3.TTF font.
4. Run SolarEye.exe
5. Load your video (or sample video).
6. The SolarEye will now trying to detect the sun as you play it. You can change the edge detection treshold and center detection treshold.
